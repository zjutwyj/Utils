1.Package Control安装
安装指导 　https://sublime.wbond.net/installation

2.插件安装
通过快捷键 Ctrl+Shift+P 打开Package Control来安装插件了。在打开的输入框中输入 install ，会根据你的输入自动提示，选择 Install Package 回车。
重启后查看是否存在 Preferences -> Package Ctroller， 则安装成功。

3.推荐插件  https://sublime.wbond.net/
1）	SideBar Enhancements　　这个插件改进了侧边栏，增加了许多功能

2）	Emmet  一种快速编写HTML/CSS代码的方法 
	- 官方文档：http://docs.emmet.io/ 
	- 速查表：http://docs.emmet.io/cheat-sheet/

3) 	BracketHighlighter 高亮显示匹配的括号、引号和标签
4)	Better Completion 代码自动提示 https://github.com/Pleasurazy/Sublime-Better-Completion
5)	DocBlockr　　DocBlockr可以自动生成PHPDoc风格的注释。它支持的语言有Javascript, PHP, ActionScript, CoffeeScript, Java, Objective C, C, C++
6)	jsFormat js代码格式化
7）	tag 	标签格式化


4.常用快捷键
1)	Ctrl+Shift+P 打开Package Control, 输入install  即可搜索并安装插件
2）	Ctrl + P   	根据文件名打开文件 当输入“ ：” + 数字 表示定位到行号
3）	Ctrl + R	查找函数方法
4）	Ctrl + G 	定位到行号
5）	Ctrl + D 	选中某个变量时  按下Ctrl+D 可以同时修改多个相同名称的变量
6)	Ctrl + F	查找内容 按F3可以定位到下一个
7）	Ctrl + Shift + F 	全文搜索 带替换功能
9）Ctrl+L 选择整行（按住-继续选择下行）
10） Ctrl+Shift+K 删除整行
11) Ctrl+D 选词 （按住-继续选择下个相同的字符串）


Ctrl+KK 从光标处删除至行尾

Ctrl+Shift+D 复制光标所在整行，插入在该行之前
Ctrl+J 合并行（已选择需要合并的多行时）

Ctrl+KL 改为小写

Ctrl+/ 注释整行（如已选择内容，同“Ctrl+Shift+/”效果）
Ctrl+Shift+/ 注释已选择内容
Ctrl+Z 撤销
Ctrl+Y 恢复撤销
Ctrl+M 光标跳至对应的括号
Alt+. 闭合当前标签
Ctrl+Shift+A 选择光标位置父标签对儿
Ctrl+Shift+[ 折叠代码
Ctrl+Shift+] 展开代码
Ctrl+KT 折叠属性
Ctrl+K0 展开所有
Ctrl+U 软撤销
Ctrl+T 词互换
Tab 缩进 自动完成
Shift+Tab 去除缩进
Ctrl+Shift+↑ 与上行互换
Ctrl+Shift+↓ 与下行互换
Ctrl+K Backspace 从光标处删除至行首
Ctrl+Enter 光标后插入行
Ctrl+Shift+Enter 光标前插入行
Ctrl+F2 设置书签
F2 下一个书签
Shift+F2 上一个书签
