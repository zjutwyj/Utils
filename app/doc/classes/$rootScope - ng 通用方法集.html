[初始化] - modalInit

  • $scope
  • $modalInstance
  • item
)

Defined in C:\software\WebstormProjects\Utils\app\scripts\app.js:147

通用弹出对话框 初始化

Parameters:

  • $scope Object

    作用域

  • $modalInstance Object

    对话框实例对象

  • item Object

    传递的数据 可以是object, 或数值

Example:

 var modalInstance = $modal.open({
     templateUrl: 'modules/Upload/views/copy_src.html',
     controller: ['$scope', '$modalInstance', function($scope, $modalInstance){
         $scope.pic = pic;
         $rootScope.modalInit($scope, $modalInstance, $scope.pic);
     }],
     windowClass : 'pic-upload-dialog',
     resolve: { pic: function () { return pic; } }
 });

modalInstance.result.then(function () { $rootScope.showMsg('复制成功'); });
